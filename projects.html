Alameer Hydraulics

E-commerce 2023

Project Overview

A custom-built mobile app for a hydraulic parts store in Palestine, developed with modern UI/UX standards and Firebase backend. The app allows customers to browse products, place orders, and track deliveries, while the admin can manage inventory and process orders.

Key Features

  • Product catalog with categories and search
  • Shopping cart and checkout process
  • Order tracking and history
  • Admin dashboard for inventory management
  • Push notifications for order updates

Technologies Used

Flutter Firebase Dart Node.js UI/UX Design
